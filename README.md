# DigiBrandBooster

DigiBrandBooster is a fast, static marketing website for a digital growth agency focused on SaaS and online revenue growth.

## Project Files

- `index.html` - main landing page
- `cases.html` - case studies page
- `styles.css` - site styles
- `script.js` - navigation and interactive behavior
- `LAUNCH.md` - deployment and launch checklist
- `LOGO_REPLACEMENT_GUIDE.md` - logo customization notes

## Local Preview

Open `index.html` directly in a browser, or use any simple static server if you want to preview the site locally.

## Deployment

This site is ready to deploy as static files to Netlify, GitHub Pages, Vercel, or any standard web host.

Before publishing, review the contact details in `index.html` and `cases.html` and replace the placeholder email and phone number with your own.

## Customization

- Update the brand copy in `index.html` and `cases.html`
- Adjust colors and layout in `styles.css`
- Edit the logo using `LOGO_REPLACEMENT_GUIDE.md`
- Follow `LAUNCH.md` for the full launch checklist
